# 👨‍💻 Portfolio Website

A personal **Portfolio Website** showcasing my skills, projects, and contact information.  
Built with **HTML, CSS, and JavaScript**, this website provides a clean and responsive interface to present my professional profile.

---

## 🚀 Features

- Responsive layout for desktop and mobile.
- Interactive navigation with links to sections:
  - Home
  - Resume Download
  - Projects
  - Skills
  - About Me
  - Contact
- Embedded video background and personal photo.
- Project showcase with descriptions and external links.
- Skill cards display technical expertise.
- Footer with LinkedIn & GitHub social links.

---

## 🎨 Technologies Used

- ✅ HTML for structure  
- ✅ CSS for layout, animations, and responsiveness  
- ✅ JavaScript for interactivity  

---

## 📋 Usage

1. Open `index.html` in any modern web browser.
2. Navigate through sections: About Me, Skills, Projects, Contact.
3. Download Resume directly from the button.
4. Visit projects via provided links.
5. Connect via LinkedIn and GitHub.
